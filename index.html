Códigos de linguagem de programação

Conceito: Códigos de linguagem de programação são conjuntos de símbolos e instruções que são usadas para criar programas de computador. Eles são o meio de comunicação entre humanos e máquinas, algo que permite os devs expressarem lógica e algoritmos através de uma sintaxe específica. Esses códigos podem ser interpretados e convertidos em linguagens de máquinas, possibilitando a execução das tarefas, especificadas pelos programadores, através das máquinas. As diferentes linguagens de programação têm suas próprias estruturas, regras e finalidades, oferecendo abrangência e flexibilidade para às diversas necessidades de desenvolvimento de software.

Exemplos de linguagens programação desktop:

  • Cobol - este é uma linguagem de programação de alto nível, sua sigla pode ser traduzida como linguagem comum orientada para negócios, foi criada há 61 anos na década de 1959, e hoje em dia ainda permanece como uma das mais utilizadas;
  • PHP - este é uma linguagem de programação de alto nível, linguagem de script, usada a favor de criar matérias em um site;
  • Assembly - este é uma linguagem de programação de baixo nível que é feita diretamente com comunicando com o hardware de um computador;
  • Ruby - esta é uma linguagem de programação de alto nível, caracterizada como uma linguagem dinâmica, open source, este tem como âmago a produtividade e a simplicidade;
  • R - este é uma linguagem de programação de alto nível, por ser simples e de graça, ele é bastante usado, e foi desenvolvida em 1995.

Exemplos de linguagens programação mobile:

  • Kotlin - é uma linguagem de programação de alto nível, que oferece suporte e uma programação orientada a objetos e funcional;
  • C# - é uma linguagem de programação de alto nível, ela é orientada a componentes e objetos, é muito usada no desenvolvimento de componentes de software;
  • JavaScript - é uma linguagem de programação de alto nível, que foi criada em 1996, esta é fácil de se usar e compatível com vária plataformas;
  • Java - é uma linguagens de programação de alto nível, esta é uma das mais famosas, por ser rápida, segura e confiável para codificar;
  • Swift - é uma linguagem de programação de alto nível, tem como objetivo ser intuitiva, foi desenvolvida pela Apple, para a criação de apps para iOS, Mac, Apple Tv e Apple Watch

Exemplos de linguagens programação web:

  • PHP - é uma linguagem de programação de alto nível, linguagem de script, usada a favor de criar matérias em um site;
  • Java - é uma linguagens de programação de alto nível, esta é uma das mais famosas, por ser rápida, segura e confiável para codificar.
  • Pyton - é uma linguagem de programação de alto nível, usada para aplicações da Web, desenvolvimento de software, ciência de dados e machine learning;
  • CSS - é uma linguagem de programação de alto nível, criada com o objetivo de ajudar na estilização de linguagens de marcação como HTML;
  • C++ - é uma linguagens de programação de alto nível, lançada em 1985 como uma extensão do C, mas logo ganhou uma identidade própria, por ser bastante versátil e se mostrar diferente do C;

Diferença em programadores de Front-end, Back-end e Full stack

Front-end: Programadores de Front-end trabalham fazendo a parte visivel do site, a parte que o publico consegue ver, ler e interagir. Algus exemplos de linguagem de programação Front-end, muito usadas são:

  • CSS
  • JavaScript

Back-end: Programadores de Back-end fazem a parte mais dificil da programação, eles fazem a parte que não é visivel, como classes de um determinado código. Algus exemplos de linguagem de programação back-end são:

  • Python
  • Java
  • Ruby
  • PHP

Full stack: Programadores de Full stack trabalham fazendo os dois últimos sitados, eles fazem tanto a parte visível quanto a parte "invisível".

O que é pseudocódigo?

Conceito: pseudocódigos são como uma mistura do código de maquina, com a lingua falada no país em que o dev se encontra. Eles são chamados assim, pois não códigos "oficiais", são usados apenas para ter uma ideia do que fazer em um programa oficial. É vantajoso introduzir os pseudocódigos para iniciantes pois eles podem servir de exemplo para como séria programar um código de verdade, além disso sua linguagem é fácil de entender, pois é normalmente na lingua materna do programador iniciante. Alguns exemplos de pseudocódigos são:

  • VisualG30
  • Portugol
